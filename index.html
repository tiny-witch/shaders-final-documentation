Depth

The depth in this case refers to the distance from tthe camera to the object. This creates some issues with more complex shapes as the values can only range from 1 to 0. As a result, the approach looses it's effectiveness in models that as large and require fine detail in the depth buffer. In this case, we are using relatively small objects so the issue is not as pronounced.

Depth with Sobel

When Sobel edge detecttion is applied to the depth buffer, we get to see the areas where depth changes drastically- ie. object outlines, holes, and protrusions. For example,

    the lid of the Utah teapot
  • the outline of the cube
  • etc..

Normal

We usually see normal materials used to desrcibe the way light bends around an object in 3d modeling. However, using the noral map in combiination with a Sobel shader results in outlines where tthe direction of the faces of the object change.

Normal with Sobel Edge

When we apply a Sobel shader to the normal material buffer we can see things like:

  • flat edges meeting flat edges (ie. the faces of the cube)
  • curved edges meeting flat edges (ie. the edges of the cylinder)

Final

To get the final result, I am using the Sobel edge detector simultaneously on both the normal and depth buffer and multipying the Sobel results to get the black lines.

Final with light

Here I have applied two seperate buffers- a normal and lit scene- in order to make an outlined lit demo.
